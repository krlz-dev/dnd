# AI Dungeon Master

A browser-based D&D campaign player powered by AI. Create characters, explore campaigns, and embark on text-based adventures with an AI Dungeon Master.

## Features

- **Character Creation** - Build your hero with races, classes, stats, and backstories
- **Campaign Library** - Choose from official, community, or AI-generated campaigns
- **Dice Roller** - Roll any dice (d4-d100) with custom roll support
- **AI Dungeon Master** - Multiple AI provider options (OpenAI, Groq, Cohere, Hugging Face, Demo mode)
- **Combat System** - Turn-based combat with attack, defend, spell, and flee options
- **Save/Export** - Export and import your game progress

## Quick Start

1. Open the app in your browser
2. Create a character or use the random generator
3. Select a campaign from the library
4. Configure your AI provider in Settings (or use Demo mode)
5. Begin your adventure!

## AI Providers

- **OpenAI** - GPT models (requires API key)
- **Groq** - Free Llama models
- **Cohere** - Free tier available
- **Hugging Face** - Free models
- **Demo Mode** - Works without any API key

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (Cinzel, Crimson Text)
